    Log-off


    Requires authentication Faz log-off do usuário.

    Example request:

    curl -X POST \
    "https://octopusfit.com.br/api/v1/auth/logout" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -H "Authorization: Bearer {token}"
    const url = new URL(
    "https://octopusfit.com.br/api/v1/auth/logout"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
    "Authorization": "Bearer {token}",
};

fetch(url, {
    method: "POST",
    headers: headers,
})
    .then(response => response.json())
    .then(json => console.log(json));

    Example response (200):

    {
    "result": true,
    "message": "Usuário deslogado!"
}

    Login

    Autenticação via e-mail e senha para obter um token JTW bearer.

    Example request:

    curl -X POST \
    "https://octopusfit.com.br/api/auth/login" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -H "Authorization: Bearer {token}" \
    -d '{"email":"contato@octopusfit.com.br","password":"123!abc"}'

    Redefinir senha

    Funcionalidade da tela inicial para redefinição de senha por esquecimento. Depois do primeiro login com a nova senha, a chave 'reset_password' ficará como true e o app deverá direcionar o usuário para uma tela onde deverá ser escolhida a senha definitiva. Para tanto, chamar o endpoint 'Alterar senha'.

    Example request:

    curl -X POST \
    "https://octopusfit.com.br/api/user/forgot-pass" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -H "Authorization: Bearer {token}" \
    -d '{"email":"contato@octopusfit.com.br"}'

    const url = new URL(
    "https://octopusfit.com.br/api/user/forgot-pass"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
    "Authorization": "Bearer {token}",
};

let body = {
    "email": "contato@octopusfit.com.br"
}

fetch(url, {
    method: "POST",
    headers: headers,
    body: body
})
    .then(response => response.json())
    .then(json => console.log(json));

    Example response (200):

    {
    "result": true,
    "message": "Foi enviado um e-mail para $email com a nova senha de acesso!"
}

    HTTP Request

    POST api/user/forgot-pass

    Body Parameters

    Parameter Type Status Description
    email string required E-mail do usuário usado para acesso ao app.
